# Udemy-Clone
A simple project built while learning web development.

A responsive frontend website built using **HTML** and **CSS (Flexbox)**.  
This project demonstrates modern layout techniques using `display: flex`, `justify-content`, `align-items`, and `flex-wrap`.

This website is a courses to know by the user in UI that includes:

- Navigation Bar  
- Categories Section  
- Recommended Courses  
- Topics Recommended  
- Popular Courses  
-  Footer

The design is structured using **CSS Flexbox** for responsive layout handling.

---

##  Technologies Used

- HTML5  
- CSS3  
- Flexbox Layout  

---

##  Website Sections

###  1. Navigation Bar
- Contains logo and menu links
- Aligned using `display: flex`
- Uses `justify-content: space-between`
- Responsive structure

### 2. Categories Section
- Displays course categories
- Uses flexbox for equal spacing
- Wraps automatically using `flex-wrap`

###  3. Recommended Courses
- Course cards layout
- Center aligned using `justify-content`
- Cards arranged using `flex-wrap` for responsiveness

### 4. Topics Recommended
- Displays trending or suggested topics
- Flex layout for alignment
- Clean spacing using `gap`

### 5. Popular Courses
- Highlighted course cards
- Responsive grid-like structure using Flexbox

###  6. Footer
- Contains contact information and links
- Structured using flexbox
- Proper alignment and spacing
